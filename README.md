📌 Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de demonstrar o funcionamento da ferramenta Sprite Shape na engine Unity.

O Sprite Shape permite criar formas e terrenos 2D de maneira dinâmica e flexível, facilitando a construção de cenários com curvas suaves e personalizadas, sem a necessidade de desenhar cada variação manualmente.

Neste projeto, exploramos como essa ferramenta pode ser utilizada para:

Criar terrenos 2D de forma procedural ou manual
Ajustar curvas e colisores automaticamente
Melhorar a eficiência na criação de cenários
